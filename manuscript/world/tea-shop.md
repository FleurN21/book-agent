# The Tea Shop

## Overview
A tea shop near the hospital where Evie received treatment. Alina and Evie went almost every Saturday for years. This location is the critical clue — the tea shop memory Parker hides, and the renovation date that breaks the case.

## Sensory Details
- A comfortable, familiar place associated with Alina's happiest memories with Evie
- The current version: has wallpaper (installed during renovation 3 years ago)
- The pre-renovation version (as seen in Parker's memory): different floors, no wallpaper
- The difference between these versions is subtle but crucial

## Story Function
- The tea shop memory: Parker's hidden memory shows Alina at the tea shop, pre-renovation
- Ch 9: Alina visits the tea shop for the first time since Evie's death. Emotional weight.
- Ch 13: Alina realises the floors/wallpaper are different from the memory → renovated 3 years ago → memory is pre-wipe → Parker was never truly wiped
- The manager confirms: renovated 3 years ago when new owner bought the place

## The Critical Clue Chain
1. In the memory: tea shop has old floors, no wallpaper
2. In reality: tea shop has new wallpaper (3 years old)
3. Parker was wiped 3 years ago
4. Therefore: the memory is from BEFORE the wipe
5. Therefore: Parker retained pre-wipe memories
6. Therefore: she was never truly wiped

## Consistency Notes
- Near the hospital where Evie was treated
- Alina and Evie went every Saturday
- Stopped going when Evie went into remission (no longer near hospital for treatment)
- Alina hasn't been back in a long time
- The manager is a balding man (bought the place 3 years ago from previous owner)
- A young server (~17 years old) also works there
- The renovation detail must be mentioned subtly when Alina first visits (Ch 9) — noted as different but not emphasised
